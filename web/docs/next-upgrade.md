# Next.js 15 Upgrade Notes

This project now targets **Next.js 15** and **React 19**. All packages have been upgraded accordingly, including `next-auth` and `next-sanity`.

If you encounter dependency conflicts when adding new libraries, double-check that the versions support Next.js 15 and React 19. You may need to update peer dependencies or use the `--legacy-peer-deps` flag during installation.
